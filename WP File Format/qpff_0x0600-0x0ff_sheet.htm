Quattro Pro File Format

Sheet Records - 0x0600-0x06FF

The following are sheet records, which span from 0x0600 to 0x06FF.

0x0601

BOS : Beginning of sheet

The data in each sheet begins with a BOS record. The stream position of each BOS is referred to in the BLOCK record. There is a corresponding EOS record.

[record type = 1537 (0x0601)]

[body length = variable]

[sheet] sheet # for all cell records to follow until EOS

[firstCol] index of first column

[lastCol] index of last column (write 0 if unknown)?

{firstRow} index of first row

{lastRow} index of last row (write 0 if unknown)?

[format] sheet default format index (-1 = use notebook default)

[flags]

bits 0-3: hidden

0 = normal

1 = hidden

2 = very hidden (only showable through Microsoft Visual Basic routine)

15 = use notebook default

bits 4-7: type

0 = normal sheet

1 = dialog sheet

2 = macro sheet

3 = object sheet

4 = Microsoft Visual Basic module

5 = chart

bits 8-15: reserved

QSTR_LABEL name sheet name (if any)

0x0602

EOS : End of sheet

[record type = 1538 (0x0602)]

[body length = 8]

{prev end} file position of previous end of sheet record (or corresponding beginning of sheet record if this is the first sheet)

{nested end} file position of the last end of column record (0 if none)

0x603

SHEET_PROTECT

Sheet protection flag - Enables or disables protection for a page.

[record type = 1539 (0x0603)]

[body length = 2]

[sheet protect]

0 = not protected

1 = protected

0x0631

SHEET_DEFAULT_ROW_HEIGHT

For rows, only write out the global height if it is an override, because on load the default font id takes precedence. Ditto for global width, only if it's different than the default for that system... that way, when users load it into another system, they see the same number of characters.

MSB indicates default is an override (does not follow font)

 

[record type = 1585 (0x0631)]

[body length = 2]

[default height]

Outline Records

Outline records are a subset of sheet records.

0x063B

OUTLINE_ROW_COL_HEADER

[record type = 1595 (0x063b)]

[body length = 4]

{flags}

bit 0: row summary before

bit 1: col summary before

bit 2: row hidden

bit 3: col hidden

bits 4-31: unused

0x063C

OUTLINE_ROW_NODE

[record type = 1596 (0x063c)]

[body length = 12]

{end node}

bit 1-31: unused

{end node}

{flags}
