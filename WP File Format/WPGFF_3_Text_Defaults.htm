WordPerfect Graphic File Format : Text Functions / WPG Defaults

WordPerfect Graphic File Format

Text Functions

The topics in this section provide the functions supported and their values.

Single-Byte Characters: 1-127 (0x01-0x7F)

Prefix character maps are not supported. Any characters not in the default character map must be encased in extended character gates

Single-Byte Functions: 128-207 (0x80-0xCF)

Soft and hard space codes - 128, 129 (0x80, 0x81) are rendered as ' '.

The soft hyphen at end of line and hard hyphen in line codes - 131, 132 (0x83, 0x84) are rendered as '-'.

The hard and soft end of center/align - 136, 137 (0x88, 0x89) codes are valid codes in WPG 7/8 text.

Hard, soft, deletable, and undeletable codes - 185, 188, 204, 207 (0xB9, 0xBC, 0xCC, 0xCF) are also valid in WPG 7/8 text.

Variable-Length Multi-Byte Functions: 208-226 (0xD0-0xE2)

Paragraph Group: 211 (0xD3)

The justification mode code, subfunction 5: Full justification may not be supported by some implementations, and its level of sophistication may differ across others.

Character Group: 212 (0xD4)

The set underline spaces mode code, subfunction 2, is fully supported.

The set underline tabs mode code, subfunction 3, is fully supported.

The horizontal advance (relative or absolute) code, subfunction 4, is fully supported (units are 1200ths, regardless of the units of measure defined in the Start WPG record; affected by graphics transformations outside the text region; absolute horizontal positions relative to left edge of text region).

The vertical advance (relative or absolute) code, subfunction 5, is fully supported. (units are 1200ths, regardless of the units of measure defined in the Start WPG record; affected by graphics transformations outside the text region; absolute horizontal positions relative to left edge of text region).

The font change code, sub-function 26 (0x1A), is fully supported except with regard to the following additional information required at the beginning of the deletable information:

<fixed-length part of WP 7/8 typeface descriptor> x 24 (for best-fit matching)

[total string length of 4 font name strings] in bytes (including all nulls)

[typeface name and pitch/point size] x ? (null-terminated WP word string)

[attribute description] x ? (null-terminated WP word string)

[extension] x ? (null-terminated WP word string)

[group] x (null-terminated WP word string)

Note: If total string length is zero, no strings or nulls follow, and the font is best-fit matched according to the values in the fixed-length part of the typeface descriptor.

The font size change code, sub-function 27 (0x1B), is fully supported except for the following additional information required at the beginning of the deletable information:

[desired horizontal point size (3600ths of an inch)] (affected by graphics transformations outside the test region)

[desired vertical point size (3600ths)] (affected by graphics transformations outside the test region)

The overstrike code, subfunction 41 (0x29), is fully supported.

Tab Group: 224 (0xE0)

The tab/indent/center/align code (subfunction zero) is fully supported for only the following tab types:

Type # Tab Type
0 (0x00) Back tab (left margin release)
16 (0x10) Left-aligned tab
48 (0x30) Left indent
56 (0x38) Left/right indent
64 (0x40) Center (on text region edges)
72 (0x48) Center on current position
80 (0x50) Center-aligned tab
128 (0x80) Flush right
144 (0x90) Right-aligned tab
208 (0xD0) Decimal-aligned tab

In addition, the following information must be supplied at the beginning of the deletable information:

[horizontal alignment position] (affected by graphics transformations outside the text region; absolute horizontal positions relative to left edge of text region)

[WP character set/index of alignment character (ignored if not decimal tab type)]

[WP character set/index of dot leader character (ignored if no dot leader)]

[# of spaces between dot leader characters (ignored if no dot leader)]

[starting position (for conversions)]

Platform Group: 225 (0xE1)

Style-rendition attribute codes, subfunctions 12-26 (0xC-0x1A), apply various rendition attributes to graphics text just as corresponding top-level WPG 2.0 rendition attribute records are applied to graphic objects. See the Rendition records in Type under Record Header earlier in this section for more information. Any of these functions that have units of measurement should use the units defined in the Start WPG record.

Fixed-Length Multi-Byte Functions: 240-247 (0xF0-0xF7)

The extended character code 240 (0xF0) is fully supported.

Attribute on and attribute off codes 242-243 (0xF2-0xF3) are supported. However, the number of attributes rendered (of the 18 possible) may differ from implementation to implementation.

Functions that may be important for editing graphics text, but can be ignored by WPG 2.0 Rendering Code

Outline Definition

The following information must be supplied at the beginning of the deletable data:

<type flag>

bit 0: set if outline on

bit 1: set if outline off

bit 2: set if paragraph number definition

(the following data exists if bit 2 above is set)

<8 new definitions (3 bytes each containing 6.0 level characters)>x24

[8 new level numbers]x8

<new attach flag>

<new outline flag>

Corel Presentations Slide Options, Presentations OrgChart Node Options

These codes are documented as 0xE11C and 0xE11D in the WordPerfect File Format section.

Soft Hyphen in Line and Block ON/OFF

No need to render these functions.

Date Format, Outline Definition, and the Paragraph Style Begin/End and Part 1/Part 2 ON/OFF codes

These functions can be ignored by rendering code, because their information should already have been extracted during formatting and inserted between begin and end function gates.

Begin/End Date Function, and Begin/End Paragraph Number

These functions themselves are ignored, but the text between them is rendered normally.

Tab Set, Decimal Alignment Character, and Dot Leader Character

These functions can be ignored by rendering code, because their information should have already been extracted during formatting and inserted in the deletable information of the TAB function.

Other 7/8 Text Functions

Any other 7/8 text functions will be preserved, but are ignored by the WPG 7/8 editing and rendering code.

WPG Defaults

A minimum WPG file contains the fixed 26-byte file prefix, a Start WPG record, and an End WPG record. No other requirements affecting file content exist. Therefore, to assure consistent performance by different products running on different platforms, the following default values are in effect in the absence of various rendition control records.
